## Project Overview

This project analyzes sales performance using Power BI.

The dashboard provides:
- Sales trend analysis
- Regional performance comparison
- Product performance
- Target vs actual sales


## Dataset

The dataset used in this project was provided by the lecturer for academic purposes.
Due to ownership and distribution restrictions, the original dataset is not included in this repository.

The Power BI dashboard file contains the data model, Power Query transformations, DAX measures, and visualizations developed during the analysis.


## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Excel
